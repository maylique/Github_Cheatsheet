GitHub

CheatSheet

git init

Initialisiert eine neue Repo

git clone

Erstellt eine Kopie einer Repo

git add < . / file >

Fügt mehr oder weniger, alles oder nur etwas, in die Pushqueue

git log

Zeigt die Logs an.
Kann mit < --oneline > oder < --graph > übersichtlicher gemacht werden

git commit -m 'Hier könnte ihre Werbung stehen'

Fügt dem push Kommentar hinzu

git push

Push den Code in die Repo
Wenn es nicht klappt evtl. mit < --force >
Bei push aus branch entweder origin angeben oder einfach aus der console copypasten

git branch < blubbelfasel >

Erstellt eine Branch abseits von main
< -b > Löscht Branch
Ohne < blabla > --> Zeigt Branches an, mit < q > gehts raus

git checkout < blabla >

Wechselt zur anderen Branch
< -b > Erstellt und wechselt zur anderen Branch

git merge < main >

Wird von der Branch ausgeführt, um mit main zu mergen
